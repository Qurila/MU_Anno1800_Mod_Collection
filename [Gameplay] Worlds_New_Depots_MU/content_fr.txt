#Worlds_New_Depots_MU



	Basé sur le dlc industrie, ce mod offre la possibilité de construire une ville de "stockage" évolutive avec des modules dockland modifiés.
	"Speicher"-city", et ainsi de créer un "district industriel".
	Il est possible de construire des dépôts sur la terre dans l'Ancien Monde, le Nouveau Monde, l'Arctique et Enbesa, qui peuvent être mis à niveau en 3 étapes.
	mise à niveau.
	
	Cependant, le mod actif "New World Docklands" de @Lrds xormenter est nécessaire pour cela :
	https://www.nexusmods.com/anno1800/mods/215
	
	En correspondance optique, 2 entrepôts avec un passage sont inclus, qui peuvent également être améliorés en 3 étapes.
	
	Tout est débloqué en même temps que la ville de stockage du DLC Docklands.
	
	Dans le Vieux Monde, vous pouvez sélectionner la peau souhaitée avec le pinceau dans le menu de construction :
	
	- Docklands Original pour l'Ancien Monde
	- Skin du Nouveau Monde issu du mod "New World Docklands".
	- Skin Enbesa du mod "New World Docklands".
	
	Comme vous ne pouvez pas changer de skin dans les autres sessions, seuls les skins du mod "New World Docklands" sont disponibles dans ces sessions.
	
	Pour l'Arctique, la texture du mod "Winter" de @Lirvan a été utilisée.
	https://www.nexusmods.com/anno1800/mods/311
	Puisqu'il nous a aussi donné la possibilité de les créer nous-mêmes avec ses outils,
	de les créer avec ses outils, je n'y vois aucun problème et je l'en remercie vivement.
	Je n'ai pas réussi à les créer moi-même :( .
	
	De plus, ce mod apporte avec lui un dépôt souterrain.
	A travers le dôme en verre, vous pouvez observer ce qui se passe sous terre.
	Dans l'ancien monde, vous pouvez sélectionner le skin souhaité dans le menu de construction avec le pinceau.
	Vous pouvez choisir entre :
	
	- Adapté aux Docklands pour l'ancien monde.
	- Adapté au mod "New World Docklands" pour le Nouveau Monde
	- Adapté au mod "New World Docklands" pour Enbesa
	
	J'ai adapté les couleurs de l'Arctique.
	
	
	Les entrepôts sont identiques aux bâtiments vanille lorsqu'ils sont améliorés :
	- LvL 1 a 2 rampes de chargement
	- Lvl 2 a 3 rampes de chargement
	- Niveau 3 : 4 rampes de chargement
	- Niveau 4 : 6 rampes de chargement
	
	Les dépôts "Dockland" ont les capacités de stockage suivantes :
	- LvL 1 a 30t
	- LvL 2 a 65t
	- LvL 3 a 105t
	- LvL 4 a 150t
	
	Les dépôts souterrains ont une capacité de stockage de 30t par objet et ne sont pas extensibles.
	
	Ces valeurs seront modifiées dans le futur avec la version finale du Mod Manager Anno 1800 :
	https://github.com/anno-mods/iModYourAnno
	librement modifiable.
	
	Vous pouvez trouver le menu de construction :
	
	-dans l'ancien monde chez les artisans, après le "middle harbour".
	-Dans le Nouveau Monde chez les Oberos, après le "Grand port".
	-Dans l'Arctique, chez les Techniciens, après le "Port
	-Enbesa, chez les Anciens, après le "Grand port".
	
	Le mod est disponible pour toutes les langues cette fois-ci ;)
